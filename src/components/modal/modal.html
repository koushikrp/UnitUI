Modal

Simple dialog box/modal

For showing modal add showModal function to the button. Add btn-show-modal to the button, for primary button. For closing the modal, add closeModal() function in button. 

							
								<button class="btn-show-modal" onclick="showModal()">

									Show Simple dialog
								</button>

								<div class="modal-wrapper">

									<div class="modal-content">

									Do you want to continue ?

									<button onclick="closeModal()" class="closeBtn">×</button>

									</div>

								</div>
