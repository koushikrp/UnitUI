Spaced list

Normal spaced list can be used with spaced-list for list and list-item for adding item in the list. Alternative version of this list is also avaialble using spaced-list-alt.

Stacked list (Notification list)

For using notification list, add stacked-list class for list. For using item in stacked list use stacked-list-item. Using image in stacked list you have to use list-img and for showing content use list-content.
