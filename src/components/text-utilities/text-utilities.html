Text utilities

Headings

Headings range from class h1 to h6 class which are modifications of default headings

h1 - This is h1 heading

h2 - This is h2 heading

h3 - This is h3 heading

h4 - This is h4 heading

h5 - This is h5 heading
h6 - This is h6 heading
							
								<h1 class="h1">h1 - This is h1 heading</h1>

								<h2 class="h2">h2 - This is h2 heading</h2>

								<h3 class="h3">h3 - This is h3 heading</h3>

								<h4 class="h4">h4 - This is h4 heading</h4>

								<h5 class="h5">h5 - This is h5 heading</h5>

								<h6 class="h6">h6 - This is h6 heading</h6>

Small text

For using small text, apply text-small class to the element

Small text

							
								<p class="text-small">Small text</p>

Gray text

For using gray text, add text-gray class to the element

Gray text

							
								<p class="text-gray">Gray text</p>

Text center

For using centered text, apply text-center class to the element

Text center

							
								<p class="text-center">Text center</p>
