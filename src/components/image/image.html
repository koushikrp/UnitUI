Image

Images with different sizes

Images with different sizes are available with class img img-sm, img-md and img img-lg for small, medium and large image

Responsive Image

Responsive images can be used with img-fluid class

Round image

For making image rounded add img-rounded class, also for making a whole circle height and width of the image should be same, for that add sm-square and md-square for small and large images
