Toast

Simple toast

For showing toast button add toast-btn class to a button, and add showToast() function to the button, for showing toast. For the animated toast itself add div with clas toast-ani. 

							
								<button class="toast-btn" onclick="showToast()">Show Toast</button>

								<div id="toast-ani">

								<span>This is a simple toast</span>

								</div>
