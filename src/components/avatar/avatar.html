Avatar

Profile Avatar

This is default profile avatar which can be added with avatar class

Avatar with different sizes

Size of the avatar can be added with avatar-sm avatar-md & avatar-lg class

Avatar with live status

Status of the avatar can be added with different live statues status-badge-busy, status-badge-offline & status-badge-online class

Avatar different shapes

Shapes of the avatar can be added with different class shapes avatar-square avatar-rounded class
