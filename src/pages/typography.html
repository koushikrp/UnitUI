Font Family

UnitUI uses Roboto and Raleway which are google fonts. For using these google fonts add following imports in the css file. 


							
								@import url('https://fonts.googleapis.com/css2?family=Roboto+Serif:wght@200;300;400;500&display=swap');


								@import url('https://fonts.googleapis.com/css2?family=Raleway:wght@300;400;700;900&display=swap');
